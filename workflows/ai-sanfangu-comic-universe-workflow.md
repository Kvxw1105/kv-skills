# AI三反骨：白卷羊 / 懒团团 / 灰策狼漫画宇宙工作流

## 1. 核心定位

把 AI、计算机、AI 应用、工作流、自动化、商业交付和 AI 时代认知，转成一个可持续生产的中文漫画 IP 宇宙。

```text
白卷羊：看懂系统 / 讲清楚
懒团团：改造系统 / 能偷懒
灰策狼：创造结果 / 会变现
```

外部口号：**讲清楚 · 能偷懒 · 会变现**。  
内部策略：**看懂系统 · 改造系统 · 创造结果**。

## 2. 白天主线

白天模式使用清新、暖白、浅薄荷绿、手绘知识漫画风。

默认 4 页结构：

1. 钩子 / 痛点
2. 误区 / 为什么失败
3. 机制 / 方法 / 工作流
4. 总结 / 金句 / 可转发点

三角色角度：

- 白卷羊：这是什么？背后的系统结构是什么？
- 懒团团：这件事怎么变成结构、模板、工作流、自动化？
- 灰策狼：这件事如何变成结果、交付、价值、交易？

## 3. 黑夜暗线

黑夜模式不是每天固定出现，而是不定期作为彩蛋和连续剧推进。

主题：**黑箱塔档案：为什么你越学 AI 越焦虑？**

概念级 Boss：

- 黑箱先生：用黑话制造焦虑、卖捷径。
- 噪音兽：垃圾信息、碎片化学习、乱喂 AI。
- 循环怪：失控自动化和无限 Loop。
- 空转巨人：努力很多但没有输出。
- 伪大师：空洞课程、认知话术、伪交付。
- 旧系统：重复劳动证明价值、信息差维持权威、黑话制造门槛。

黑夜内容必须回到知识锚点：Context、Loop、Workflow、RAG、Agent、交付、问题定义、系统思维。

## 4. 互怼与联动

三账号不要只是平铺互补，要有戏剧张力。

接力互怼：

```text
白卷羊：先讲清楚是什么。
懒团团：知道是什么不如知道怎么用。
灰策狼：会用还不够，要知道怎么变成结果。
```

典型组合：

- 白卷羊讲 Agent 是目标 + 工具 + 上下文 + 循环。
- 懒团团怼：我只关心它能不能替我整理周报/文献/选题。
- 灰策狼怼：会自动化还不够，客户为可验收结果付钱。

遇到 Boss 时三人合作：

- 白卷羊翻译黑话。
- 懒团团修流程。
- 灰策狼查价值链和交付漏洞。

## 5. 视频工作流

给 10 秒 AI 视频模型时，不要让它自由创作动画片。

核心提示：**像一张会动的漫画页，而不是一段动画片。**

推荐形式：

- 三格定型板：完整三格保留，按 0-2s / 2-6s / 6-9s / 9-10s 动。
- 漫画自动下滑：适合文字密度较高页面。
- 重点格放大：适合爆点格。
- 白卷纸张变答案：适合白卷羊固定片头。
- 案件推理板：适合黑夜和拆局。
- 低帧率掉帧动态漫：8-12fps 手翻漫画节奏。

## 6. 第一批内容

推荐第一期：

- 白卷羊：为什么你问 AI，它总是答得很空？
- 懒团团：别把资料一股脑丢给 AI
- 灰策狼：只会用 AI，为什么赚不到钱？

推荐同题三拆：

- Context Engineering
- AI Agent
- RAG
- Workflow
- OPC

## 7. 质量红线

- 不要再使用“小小羊”作为主角名； canonical name 是“白卷羊”。
- 懒团团眼睛统一为暖茶棕半眯眼，禁止纯黑空洞眼和随机绿眼。
- 灰策狼不是恐怖反派，是清醒、现实、会算账的结果派。
- 现实认知选题可以做，但必须回到 AI / 系统 / 工作流 / 结果中的至少一个锚点。
- 图像文字要短，重要文本建议后期叠字。
