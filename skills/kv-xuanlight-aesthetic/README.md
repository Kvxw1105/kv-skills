# kv-xuanlight-aesthetic · 玄光美学

> **A creative style system for turning ordinary image ideas into the Xuanlight Aesthetic.**  
> Dark worlds, sacred subtle light, solitary subjects, negative space, low-resolution grain, and poetic restraint.

---

## What it is

Most image prompts stop at subject matter: *a swordsman on a mountain*, *an AI soul in a ruined city*, *a cover image with moonlight*.

`kv-xuanlight-aesthetic` goes one level higher. It does not just polish a prompt — it **translates a visual idea into a complete style system**.

It helps an Agent decide:
- which Xuanlight branch fits best,
- what emotional DNA and sacred light logic should dominate,
- how sparse or narrative the composition should be,
- and how to output reusable prompt layers rather than one-off prompt fragments.

## Best for

- visual KV / hero images
- covers / posters / banners
- character concept images
- worldbuilding art
- story illustrations
- comics / storyboard panels
- aesthetic system prompts for long-term reuse

## Core capabilities

| Layer | What it does |
|------|------|
| **Style translation** | Converts ordinary visual ideas into the Xuanlight visual language |
| **Branch selection** | Chooses between Ink / Pilgrimage / Soulcore / Ruins / Room / Panels |
| **3-layer prompt output** | Produces abstract / generic / concrete prompts in CN + EN |
| **Creative structure** | Adds negative prompts, parameter suggestions, composition hints, and iteration directions |
| **Series consistency** | Helps keep a multi-image or comic sequence stylistically coherent |

## Output structure

A typical run should produce:

1. style interpretation
2. branch selection
3. bilingual keywords
4. abstract prompt (CN / EN)
5. generic prompt (CN / EN)
6. concrete prompt (CN / EN)
7. negative prompt
8. parameter suggestions
9. size / composition guidance
10. iteration directions
