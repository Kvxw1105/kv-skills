# Quick Start｜玄光美学 Skills 包快速开始

## 30 秒理解

玄光美学不是一句固定提示词，而是一种统一视觉语言：

- 暗
- 空
- 光
- 孤
- 低清颗粒
- 神性气场
- 东方留白 / 精神科幻 / 梦境叙事

## 最快使用方式

直接套用下面的句式：

### 中文快速模板
```text
请将【主题】转译为玄光美学风格，偏向【分支名】，情绪为【情绪关键词】，场景为【场景关键词】，光源为【光源关键词】，画面简繁程度为【极简 / 简约 / 中度叙事】，用途为【封面 / 海报 / 漫画 / KV / 角色设定】。请输出：1）风格理解；2）中英双语关键词；3）抽象版提示词；4）通用版提示词；5）具体版提示词；6）负面词；7）参数建议；8）后续迭代方向。
```

### English quick template
```text
Translate [subject] into the Xuanlight Aesthetic, leaning toward [branch], with the mood of [mood keywords], the scene of [scene keywords], the sacred light source of [light keywords], a [minimal / balanced / narrative] visual density, and the usage of [cover / poster / comic / KV / character concept]. Output: 1) style interpretation; 2) bilingual keywords; 3) abstract prompt; 4) generic prompt; 5) concrete prompt; 6) negative prompt; 7) parameter suggestions; 8) iteration directions.
```

## 三种常用姿势

### 1. 做单张图
- 用 `PROMPT_LIBRARY.md` 里的“抽象 + 具体”组合。

### 2. 做系列图
- 先固定：分支、情绪、光源、色谱、简繁等级。
- 再每张只改：主体、镜头、场景动作。

### 3. 做漫画 / 剧情
- 用 `玄光·漫画` 分支。
- 一格只承载一个情绪核心。
- 统一角色外观与光源逻辑。
