# Design System: 小小羊讲AI

## Visual DNA

- Format: vertical 9:16 social-media carousel page.
- Mood: cute, clear, low-pressure, internet-friendly, educational.
- Line: thick black outline, slightly shaky hand-drawn stroke, rounded corners.
- Background: warm off-white or very light gray.
- Panels: pale mint / pale teal horizontal rectangles.
- Accent colors: soft yellow sparkles, muted red/green for the lamb's eyes, light blue/gray robot icons, small peach blush.
- Texture: flat color, no realistic rendering, no heavy shadows, no complex gradients.

## Layout Formula

1. Top cloud-like title bubble: `小小羊讲AI`.
2. Small yellow pill label: `第X页 / 共Y页`.
3. Big subtitle: topic name, e.g. `AI Agent`.
4. Three stacked horizontal panels.
5. Caption under every panel.
6. Use arrows, dotted paths, small icons, and simple props to explain causality.

## Mascot: 小小羊

- Original lamb, not a copy of any existing IP.
- White wool with rounded fluffy silhouette.
- Curled forelock as signature hair feature.
- One eye red, one eye green.
- Small rounded body, soft paws/hooves, curled tail.
- Simple facial expressions: curious, confused, excited, teaching.
- Repeat the eye colors and forelock in every prompt to maintain identity.

## Icon Language

Use simple objects as metaphors:

- AI basics: black box, word tokens, arrows, chat bubbles.
- Prompt engineering: role card, target, document format, lock/constraint.
- Agent: robot, search icon, document, calendar, memory note, play/execution button.
- Context: document stack, memory card, sticky note.
- Loop: circular arrows, iteration steps.
- Harness: gear, pipeline, control panel.
- OPC/personal company: laptop, mini robot team, notes/course, wallet/payment, audience/community.

## Caption Voice

Tone: direct, friendly, slightly playful, but not childish.

Good patterns:

- “其实，X更像Y。”
- “当它能A、B、C，它就更像D。”
- “不是为了炫技，而是为了把结果稳定做出来。”
- “你说得越清楚，AI通常就答得越像样。”

Avoid:

- Long academic definitions.
- Empty motivational copy.
- Too many English technical terms in one caption.
- Overly dense speech bubbles.

## Differentiation From Reference Images

Keep the broad category style but change the original elements:

- Use lamb mascot instead of dog mascot.
- Use AI/business education topics instead of Web3/finance only.
- Use recurring visual systems such as red-green eyes and AI mini-team.
- Use distinct metaphors and page scripts.
- Do not copy exact panel compositions, captions, account names, or UI elements.
