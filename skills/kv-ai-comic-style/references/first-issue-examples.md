# First Issue Examples

## 白卷羊: 为什么你问 AI，它总是答得很空？

Core point: AI often gives vague answers because the user gives vague goals and insufficient context.

Page arc:
1. Hook: user asks “帮我写点东西”, AI gives vague answer.
2. Misconception: not AI stupid; input is too empty.
3. Mechanism: Prompt = task; Context = what AI sees.
4. Takeaway: 清楚目标 + 具体上下文 + 输出格式 = 更好的回答.

Key quote: “AI 不一定笨，很多时候是你给的信息太少。”

## 懒团团: 别把资料一股脑丢给 AI

Core point: AI does not fear many materials; it fears unstructured input.

Page arc:
1. Hook: 懒团团 dumps PDFs, chat logs, meeting notes, screenshots, links into AI. Quote: “你以为你在喂资料，其实是在喂噪音。”
2. Misconception: AI returns a messy vague summary. Quote: “AI 不怕资料多，怕资料没结构。”
3. Method: sort materials into 背景 / 目标 / 事实 / 输出要求. Quote: “先整理输入，再追求输出。”
4. Takeaway: structured input creates checklist/table/draft. Quote: “偷懒的第一步，不是少做，是别乱做。”

Secondary/later workplace issue: “用 AI 把周报变成 30 秒任务”. Do not make the entire 懒团团 account only about weekly reports.

## 灰策狼: 只会用 AI，为什么赚不到钱？

Core point: tool ability is not commercial value; customers pay for results and delivery.

Page arc:
1. Hook: many people say “我会 AI”. Client asks “所以你能帮我解决什么？”
2. Misconception: skill list is not a product.
3. Method: ability -> delivery -> solved problem.
4. Takeaway: “市场不为会用工具付钱，只为解决问题付钱。”

Key quote: “会用 AI 是能力，能交付结果才是商品。”
